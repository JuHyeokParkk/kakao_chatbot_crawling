# 😍 kakao_chatbot_crawling
[카카오 챗봇(Crawling)] 비트코인 시세 등을 알려주는 챗봇입니다.


# 💪 구현 기능
- 비트코인 시세, 관련 기사 크롤링
- 해외 주식 티커 입력을 통한 시세 파악
- 네이버 데이터랩 크롤링
- 코스피, 나스닥 지수 파악
